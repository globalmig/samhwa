# 시트 1. 기본 DB (관련 파일 설명)

이 시트는 이 파일과 함께 쓰이는 상위 원본 파일(`00. 2026년 KEIT 배정 및 수수료 산정` 등)의 쉬트 구성과 각 쉬트에서 어떤 열(column)을 값복사/수식 처리하는지 설명하는 메모입니다. 두 개의 하위 파일(`01. 2026 KEIT 수수료 산출내역 양식`, `02. 2026 KEIT 안내공문 양식`)은 이 원본 파일에서 산출된 값을 값복사하여 PDF를 자동 생성하는 매크로 파일입니다.


## 1. "RCMS(연차별기관별_연구비 집행)(DB)" 시트

- RCMS에서 연차별·기관별 연구비 자료 다운로드 (전문기관에서 과제 배정 받을 때마다 다운로드)
- M열 ~ AR열: RCMS 값 복사
- AF열: 수식 적용 → 현금사업비 총액 합산
- AS열: 수식 적용 → 이번 배정 대상 여부 확인
- A열 ~ L열: 수수료 산정을 위한 기본 조건 설정
- B열: O열 값복사 후 숫자로 변환


## 2. "RCMS(단계기관별)(DB)" 시트

- RCMS에서 단계별·기관별 연구비 자료 다운로드 (전문기관에서 과제 배정 받을 때마다 다운로드)
- D열 ~ BV열: RCMS 값 복사
- A열 ~ C열: 수수료 산정을 위한 기본 조건 설정
- A열: F열 값복사 후 숫자로 변환


## 3. "수행기관 등록 현황(삼화)" 시트

- 삼화전산시스템에서 수행기관 목록 최신 자료 다운로드
- 다운로드만 하고 별도 수정 작업 없음


## 4. "수수료체계" 시트

- 현재 한국산업기술기획평가원(KEIT) 수수료 체계 적용
- 기타 전문기관 수수료체계는 별도 시트 참조


## 5. "담당자연락처" 시트

- 전문기관으로부터 자료를 받아 2~3개월 주기로 업데이트


## 6. "정산면제리스트" 시트

- 정부 발표 내용에 따라 비정기적으로 업데이트
- I열, J열에만 수식 적용


## 7. "자율성트랙" 시트

- 정부 발표 내용에 따라 비정기적으로 업데이트
- F열 ~ Q열까지가 실제 활용하는 정보


## 8. "삼화시스템" 시트

- 삼화전산시스템에서 과제현황 최신 자료 다운로드
- 전체 전문기관 현황을 한번에, 또는 전문기관별로 다운로드 가능
- 현재 파일은 전문기관 중 KEIT 현황만 다운로드
- F열 ~ CG열: 값복사
- A열 ~ E열: 수수료 산정을 위한 기본 조건 설정
- B열: A열 값복사, C열: A열 값복사 후 숫자로 변환


## 9. "삼화수수료 발행내역" 시트

- 삼화전산시스템에서 수수료 발행현황 최신 자료 다운로드
- C열 ~ AR열: 값복사

# 시트 2. 전문기관별 수수료 산정 흐름

이 시트는 (A) 모든 전문기관에 공통 적용되는 수수료 산정 규칙 설명, (B) 전문기관별 기관구조 흐름도, (C) 전문기관별 세부 규정·산출비율표·계산 예시로 구성되어 있습니다. 공통 규칙 설명(협약관계, 업무방식, 정산방식, 과제종류 등)은 원본 파일에서 KEIT 열에만 서술되어 있고, 다른 전문기관은 별도 언급이 없는 한 동일 규칙이 적용됩니다(원본에 '좌동'으로 표기).

## 기관구조 흐름도 (도형 기반)

원본 시트에 그려진 조직도(사각형 도형 + 연결선)를 텍스트로 재구성한 것입니다. 모든 전문기관 공통 구조: `전문기관 → 수행기관(주관기관 1개 + 공동기관 0~N개)`이며, 일부 기관은 주관기관이 별도의 위탁기관에 사무를 재위탁하는 구조가 추가됩니다.

### 한국산업기술기획평가원 (KEIT)

```
한국산업기술기획평가원
├─ 주관기관 A
├─ 공동기관 B
├─ 공동기관 C
└─ 공동기관 D
```

### 정보통신기획평가원 (IITP)

```
정보통신기획평가원
├─ 주관기관 A ─→ 위탁기관 E
├─ 공동기관 B
├─ 공동기관 C
└─ 공동기관 D
```

### 한국임업진흥원 (KOFPI)

```
한국임업진흥원
├─ 주관기관 A ─→ 위탁기관 E
├─ 공동기관 B
├─ 공동기관 C
└─ 공동기관 D
```

### 농촌진흥청 - 일반기관 트랙 (RDA1)

```
농촌진흥청
├─ 주관기관 A (일반기관) ─→ 위탁기관 E
├─ 공동기관 B
├─ 공동기관 C
└─ 공동기관 D
```

### 농촌진흥청 - 소속기관 트랙 (RDA2)

```
농촌진흥청
├─ 주관기관 A (농촌진흥청 또는 소속기관) ─→ 위탁기관 E
├─ 공동기관 B
├─ 공동기관 C
└─ 공동기관 D
```

(참고: 한국에너지기술평가원(KETEP)은 원본 시트에 별도 조직도 도형이 없으며, KEIT와 동일한 '주관기관 A + 공동기관 B/C/D' 구조에 위탁기관 없이 좌동 처리됩니다.)

## 기관별 상세 내용 (공통 규칙 + 기관별 세부표·계산 예시)

아래는 각 전문기관 열(column) 블록에 있는 모든 셀 내용을 행(row) 순서대로 재구성한 것입니다. 단일 항목은 글머리 기호(bullet)로, 표 형태 데이터는 마크다운 표로 표현했습니다. 열 문자(B, C, D…)는 원본 엑셀의 실제 열 위치이며, 원본과의 대조·검증을 위해 그대로 남겨두었습니다.
